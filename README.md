# КП Туманообразование

Мини-приложение для подготовки КП по основным параметрам:

- номер КП;
- объект;
- город;
- количество форсунок;
- длина трубки.

Номер КП можно вводить вручную или менять кнопками `−` / `+` рядом с полем номера.

Кнопка `PDF файл` формирует готовое КП в PDF. Если браузер умеет копировать PDF-файлы в буфер, файл копируется; если нет — скачивается.

Кнопка `DOCX Файл Word` формирует редактируемый Word-файл КП с таблицей, логотипом и печатью.

Дата подставляется автоматически. Количество фитингов равно количеству форсунок. Насос выбирается по количеству форсунок:

- до 20 форсунок — 1 л, 280 000 ₸;
- до 40 форсунок — 2 л, 300 000 ₸;
- до 55 форсунок — 3 л, 320 000 ₸;
- до 100 форсунок — 4 л, 360 000 ₸;
- свыше 100 форсунок — 5 л, 400 000 ₸.

## Запуск на Mac

```bash
python3 -m http.server 4173
```

Открыть:

```text
http://localhost:4173/kp_fog_app/
```

## iPhone

Для постоянной установки на экран «Домой» разместите папку `kp_fog_app` на HTTPS-хостинге, например GitHub Pages, Netlify или своем сайте. Затем откройте адрес на iPhone в Safari и выберите «Поделиться» → «На экран Домой».

Для быстрой проверки с iPhone можно открыть адрес Mac в одной Wi-Fi сети:

```text
http://IP_АДРЕС_MAC:4173/kp_fog_app/
```

Кнопка `WhatsApp` формирует один PDF-файл, скачивает его и открывает WhatsApp с готовым текстом. Затем прикрепите скачанный PDF вручную. Такой режим выбран, чтобы iPhone/браузер не дублировал файл через системное меню «Поделиться».

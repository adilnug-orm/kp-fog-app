# КП Туманообразование

Мини-приложение для подготовки КП по основным параметрам:

- номер КП;
- объект;
- город;
- количество форсунок и длина трубок для высокого давления;
- количество форсунок и длина трубок для низкого давления;
- тип предложения: высокое давление, низкое давление или оба варианта.

Если отмечены обе галочки, приложение формирует два коммерческих предложения: первое с указанным номером КП, второе со следующим номером. Например, при номере `64` будут сформированы КП `64` и `65`.

Номер КП можно вводить вручную или менять кнопками `−` / `+` рядом с полем номера.

Кнопка `PDF файл` формирует готовое КП в PDF. Если выбраны оба типа, оба КП попадают в один PDF-файл.

Кнопка `DOCX Файл Word` формирует редактируемый Word-файл КП с таблицей, логотипом и печатью. Если выбраны оба типа, оба КП попадают в один Word-файл с разрывом страницы.

Дата подставляется автоматически. Количество фитингов равно количеству форсунок.

Для высокого давления насос выбирается по количеству форсунок:

- до 20 форсунок — 1 л, 280 000 ₸;
- до 40 форсунок — 2 л, 300 000 ₸;
- до 55 форсунок — 3 л, 320 000 ₸;
- до 100 форсунок — 4 л, 360 000 ₸;
- свыше 100 форсунок — 5 л, 400 000 ₸.

Для низкого давления используется фиксированный насос 4 бара. Все цены можно менять в блоке `Цены и формулы`.

## Запуск на Mac

```bash
python3 -m http.server 4173
```

Открыть:

```text
http://localhost:4173/kp_fog_app/
```

## iPhone

Для постоянной установки на экран «Домой» разместите папку `kp_fog_app` на HTTPS-хостинге, например GitHub Pages, Netlify или своем сайте. Затем откройте адрес на iPhone в Safari и выберите «Поделиться» → «На экран Домой».

Для быстрой проверки с iPhone можно открыть адрес Mac в одной Wi-Fi сети:

```text
http://IP_АДРЕС_MAC:4173/kp_fog_app/
```

Кнопка `WhatsApp` формирует один PDF-файл и сначала пробует передать его через системное меню отправки, чтобы на iPhone можно было выбрать WhatsApp уже с прикрепленным PDF. Если браузер не поддерживает отправку файлов, PDF скачивается, затем открывается WhatsApp с готовым текстом.

После запуска отправки номер КП автоматически увеличивается на количество сформированных КП. Имя файла короткое: `КП <название объекта>.pdf`.
